\begin{abstract}
  In diesem Programmentwurf wird das Problem \emph{Erlernen von Kundenpräferenzen in einer Wohnungsbörse}
  bearbeitet. Zunächst werden die gegebenen Daten analysiert. Diese Analyse geht anschließend in das Konzept für ein
  Neuronales Netz und in ein Konzept für einen Entscheidungsbaum ein. 
  Der Entscheidungsbaum wird zusätzlich implementiert und optimiert.
\end{abstract}
